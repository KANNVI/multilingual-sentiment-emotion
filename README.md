# 🌍 Multilingual Sentiment & Emotion Analysis

Detects sentiment and emotions from text in 100+ languages.

## Stack
- XLM-RoBERTa (HuggingFace)
- TensorFlow / Keras
- scikit-learn

## Tasks
- Sentiment: Positive / Neutral / Negative  
- Emotion: Joy, Anger, Fear, Sadness, Surprise, Disgust
